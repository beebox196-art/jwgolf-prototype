# JW Golf Scripts

Automation and utility scripts for the JW Golf project.

## Available Scripts

### setup.sh
Environment setup script for new developers.

```bash
./Scripts/setup.sh
```

### build.sh
Build the Next.js application for production.

```bash
./Scripts/build.sh
```

### deploy.sh
Deploy to Vercel.

```bash
./Scripts/deploy.sh
```

## NPM Scripts

The project also uses npm scripts defined in `package.json`:

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run start    # Start production server
npm run lint     # Run ESLint
```
