# JW Golf Tests

Test files for the JW Golf website prototype.

## Test Structure

Tests follow the project naming convention:
- `page.test.tsx` — Homepage tests
- `about.test.tsx` — About page tests
- `lessons.test.tsx` — Lessons page tests
- `contact.test.tsx` — Contact page tests

## Running Tests

```bash
npm test
```

## Test Coverage Goals

- Unit tests: 80%+
- Integration tests: Key user flows
- E2E tests: Booking flow, lead capture
