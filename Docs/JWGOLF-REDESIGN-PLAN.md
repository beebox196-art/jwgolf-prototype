### Pretext Integration

**What Pretext Offers:**
Pretext is a text measurement library that enables dynamic, precise text layouts — critical for a brand built on "clarity."

**Use Cases for JW Golf:**

1. **Dynamic Hero Typography:**
   - Measure headline text to perfectly fit container
   - Responsive scaling without reflow
   - Maintain visual hierarchy across devices

2. **Pull Quote Layouts:**
   - "One Clear Fix" quotes that break out of grid
   - Precise text wrapping around images
   - Magazine-style editorial layouts

3. **Testimonial Cards:**
   - Consistent card heights regardless of quote length
   - Smart truncation with "read more" for longer testimonials
   - Balanced multi-column layouts

4. **Service Page Typography:**
   - Benefit statements that align perfectly
   - Pricing tables with optimal text spacing
   - Feature lists with consistent line heights

5. **Email Template Generation:**
   - Newsletter layouts with measured text blocks
   - Consistent rendering across email clients
   - Optimized preview text lengths

**Implementation:**
- Use pretext for all editorial content areas
- Pair with CSS Grid for layout control
- Cache measurements for performance
- Fallback to standard CSS if JS disabled

## Section 4: Hosting & Tech Stack Options

### Free Prototype Hosting

**Option 1: Vercel (Recommended)**
- **Pros:** Serverless, global CDN, excellent performance, free SSL, Git integration
- **Cons:** Serverless function limits on free tier
- **Best for:** Next.js/React sites, fast prototyping
- **Cost:** Free tier generous for prototype

**Option 2: Netlify**
- **Pros:** Excellent DX, form handling, branch previews, generous free tier
- **Cons:** Build time limits on free tier
- **Best for:** Static sites, JAMstack
- **Cost:** Free tier sufficient for prototype

**Option 3: GitHub Pages**
- **Pros:** Simple, integrated with Git, free
- **Cons:** Static only, no server-side functions
- **Best for:** Simple static sites
- **Cost:** Free

**Recommendation:** Start with **Vercel** for the prototype — best performance, modern stack, easy deployment.

### Recommended Final Hosting

**Option 1: Vercel Pro ($20/month)**
- Team collaboration
- Analytics
- Password protection for staging
- Priority support
- **Best for:** Modern React/Next.js site

**Option 2: Cloudflare Pages ($5-20/month)**
- Excellent global CDN
- DDoS protection
- Analytics
- **Best for:** Performance-focused, global audience

**Option 3: Traditional Hosting (SiteGround, WP Engine)**
- **Pros:** Familiar, WordPress optimized
- **Cons:** Slower, more expensive, dated stack
- **Best for:** If James insists on WordPress

**Recommendation:** **Vercel Pro** — modern, fast, cost-effective, scales with traffic.

### CMS vs Static Site

**Static Site (Recommended):**
- **Pros:** Fastest performance, lowest cost, highest security, easy deployment
- **Cons:** Requires developer for content updates
- **Best for:** Content that doesn't change frequently

**Headless CMS (Contentful, Sanity, Strapi):**
- **Pros:** James can update content, still fast
- **Cons:** Additional cost ($99+/month for team), complexity
- **Best for:** Frequent content updates, multiple contributors

**WordPress (Current):**
- **Pros:** James knows it, plugins available
- **Cons:** Slow, security issues, plugin bloat, dated
- **Best for:** If change resistance is high

**Recommendation:** Start static with hardcoded content. If James needs to update frequently, migrate to **Sanity** or **Contentful** headless CMS.

### Booking System Integration

**Option 1: Calendly (Recommended)**
- **Pros:** James likely already uses it, embeddable, handles time zones
- **Cons:** Limited customization, monthly cost for features
- **Cost:** Free tier available, Pro $10/month
- **Integration:** Simple embed, passes UTM params

**Option 2: Acuity Scheduling**
- **Pros:** More customization, payment integration
- **Cons:** More expensive
- **Cost:** $16+/month

**Option 3: Custom Booking**
- **Pros:** Full control, on-brand
- **Cons:** Development time, maintenance
- **Cost:** Development time only

**Recommendation:** **Calendly** for immediate launch. Consider custom booking system once revenue justifies it.

### Email Marketing Platform

**Option 1: ConvertKit (Recommended)**
- **Pros:** Built for creators, excellent automation, landing pages
- **Cons:** More expensive at scale
- **Cost:** Free up to 1,000 subscribers, then $15+/month

**Option 2: Mailchimp**
- **Pros:** Well-known, generous free tier
- **Cons:** Getting more expensive, complex automation
- **Cost:** Free up to 500 subscribers

**Option 3: Beehiiv**
- **Pros:** Modern, newsletter-focused, growth tools
- **Cons:** Newer platform
- **Cost:** Free up to 2,500 subscribers

**Recommendation:** **ConvertKit** — best for coaching business, automation sequences, and scaling.
