# JW Golf Design Documents

Architecture, UX/UI specs, and technical designs.

## Brand Guidelines

### Colors
- **Deep Green:** #1B4332 (golf course, growth, resilience)
- **Rich Navy:** #1A365D (professional, trustworthy, premium)
- **Warm Cream:** #F5F5DC (approachable, clean backgrounds)
- **Epilepsy Purple:** #7B3FA0 (subtle accent — use sparingly)
- **Gold:** #C9A227 (premium touch, success)

### Typography
- **Headlines:** Inter or Montserrat
- **Body:** Inter (16-18px minimum for accessibility)
- **Accent/Data:** JetBrains Mono (for Trackman stats)

## Technical Stack

- **Framework:** Next.js 15 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Hosting:** Vercel (recommended)

## Performance Targets

- LCP < 2.5s
- FID < 100ms
- CLS < 0.1
- Lighthouse 90+
