# JW Golf Website Prototype

The official website prototype for James Whittaker, PGA Professional - "The Resilient Coach"

## About

This is a stunning, modern Next.js website prototype built to showcase James Whittaker's unique value proposition:

- **The One Clear Fix Method** - Simplicity in a world of golf instruction overload
- **The Resilience Story** - Five brain surgeries, now coaching 50+ European Tour pros
- **Credibility** - Titleist Ambassador, Leeds & District Union Head Coach

## Features

- Dark mode with premium gold/green accents
- Fully responsive mobile-first design
- Oauick animations and scroll reveals
- Fast loading static site
- Modern Next.js 15 with App Router
- Tailwind CSS for styling

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Modern utility-first CSS
- **Vercel** - Free hosting with global CDN

## Pages

1. **Home** - Hero, credibility badges, "One Fix" explanation, testimonials, CTA
2. **About** - James's story (epilepsy journey), career highlights
3. **Lessons** - Service offerings with clear pricing tiers
4. **Contact/Book** - Lead capture form, booking integration

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

### Vercel (Recommended)

1. Push your code to GitHub
2. Import your repository to [Vercel](https://vercel.com)
3. Deploy with free tier

The site will be live with a Vercel subdomain, and you can add a custom domain.
